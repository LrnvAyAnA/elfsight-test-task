# Characters List

*[Открыть приложение](https://elfsight-test-task-deploy.vercel.app/)*

## Запуск локально
### Клонируйте репозиторий и перейдите в папку проекта:
```git clone https://github.com/LrnvAyAnA/elfsight-test-task.git```

```cd elfsight-test-task```

### Установите зависимости
```npm install```
или
yarn

### Запустите проект:
```npm start```
или
yarn start

### Проект будет доступен в браузере по адресу:
```http://localhost:3000```
